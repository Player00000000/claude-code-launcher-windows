▣ TMUX CHEATSHEET
⚡ ESSENTIAL (ใช้ทุกวัน)
Ctrl-bPrefix key — กดก่อนทุก shortcut
Ctrl-b dDetach — ออกจาก session (session ยังทำงานอยู่)
Ctrl-b %แบ่ง pane ซ้าย/ขวา (vertical split)
Ctrl-b "แบ่ง pane บน/ล่าง (horizontal split)
Ctrl-b ←↑→↓เลื่อนไปยัง pane ต่างๆ
Ctrl-b zZoom pane — ขยาย/ย่อ pane เต็มจอ
Ctrl-b cสร้าง window ใหม่
Ctrl-b n / pไป window ถัดไป / ก่อนหน้า
Ctrl-b 0–9ไป window หมายเลขนั้นตรงๆ
🖥 SESSIONS (บริหาร session)
tmuxเปิด session ใหม่
tmux new -s nameเปิด session พร้อมตั้งชื่อ
tmux lsดูรายการ session ทั้งหมด
tmux aAttach กลับ session ล่าสุด
tmux a -t nameAttach เจาะจง session ตามชื่อ
tmux kill-session -t nameลบ session
Ctrl-b $เปลี่ยนชื่อ session ปัจจุบัน
Ctrl-b sเลือก session แบบ interactive
Ctrl-b ( / )ไป session ก่อนหน้า / ถัดไป
⬛ PANES (จัดการหน้าต่างย่อย)
Ctrl-b xปิด pane ปัจจุบัน
Ctrl-b qแสดงหมายเลข pane
Ctrl-b {ย้าย pane ไปซ้าย/บน
Ctrl-b }ย้าย pane ไปขวา/ล่าง
Ctrl-b Spaceสลับ layout (even-h, even-v, tiled…)
Ctrl-b Ctrl-←→↑↓ปรับขนาด pane
Ctrl-b !แยก pane ออกเป็น window ใหม่
🗂 WINDOWS (แท็บใน session)
Ctrl-b wเลือก window แบบ visual
Ctrl-b ,เปลี่ยนชื่อ window
Ctrl-b &ปิด window (ต้องยืนยัน)
Ctrl-b lกลับ window ก่อนหน้า (last)
Ctrl-b fค้นหา window จากชื่อ
📋 COPY MODE (เลื่อน/copy output)
Ctrl-b [เข้า copy mode — เลื่อนดู output เก่าได้
qออกจาก copy mode
/ หรือ ?ค้นหาข้อความ (ใน copy mode)
Spaceเริ่มเลือกข้อความ
EnterCopy ข้อความที่เลือก
Ctrl-b ]Paste ข้อความที่ copy ไว้
PgUp / PgDnเลื่อนหน้าใน copy mode
⚙ CONFIG (~/.tmux.conf)
set -g mouse onเปิดใช้เมาส์ (คลิก/scroll/resize)
set -g prefix C-aเปลี่ยน prefix เป็น Ctrl-a (แบบ screen)
set -g history-limit 50000เพิ่ม scroll buffer
bind r source-file ~/.tmux.confReload config ด้วย Ctrl-b r
Ctrl-b :เปิด command prompt ใน tmux
Ctrl-b ?ดู shortcut ทั้งหมดใน tmux
